# Eutopia

The website for Eutopia — a good place, not a perfect one.

A static single-page site deployed via GitHub Pages to [eutopia.global](https://eutopia.global).

- `index.html` — the entire site (self-contained: inline CSS, inline JS, Google Fonts)
- `404.html` — styled not-found page
- `CNAME` — custom domain
- `.nojekyll` — serve files as-is without Jekyll processing

To work on it locally, just open `index.html` in a browser — there is no build step.
